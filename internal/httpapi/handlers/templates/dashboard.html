Telegram alerts

Send a Telegram message when a workflow run finishes.

How to set this up
  1. Create a bot. In Telegram open @BotFather → send /newbot → pick a name and username. BotFather replies with a token like 123456:ABC-def....
  2. Save the token. Paste it into Bot token below and click Save token.
  3. Pick where alerts go, then message the bot there:
    • Direct messages — open your new bot and tap Start (sends /start).
    • A group — add the bot to the group, then send /start@your_bot_username in that group. (A bot can’t read normal group messages by default — send the command form so it sees it.)
  4. Click Connect chat. It reads your most recent message to the bot and locks onto that chat. The status shows connected: <chat>.
  5. Choose Notify on (all runs / failures only), tick Enabled, then Send test to confirm a message arrives.

Order matters: send the message to the bot before clicking Connect. If it says “no messages found”, message the bot again and retry. Channels aren’t supported — use a direct chat or a group.
