Click below to create a GitHub App scoped to {{.BaseURL}}. You'll be redirected to GitHub to confirm permissions; after creation, GitHub sends you back here to finish bootstrap.

On GitHub's "Create App" page, leave the name as-is. Renaming changes the slug and breaks the install shortcut shown on the next step (gharp itself keeps working — the link just becomes stale).
If you see "name is already taken", you have a previous gharp App for this URL. Delete the old App first, then come back and click again.
